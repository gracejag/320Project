Handling Outliers

Extreme values were moderated using an inter quartile capping rule that replaces observations lying beyond 1.5 times the IQR with the nearest boundary. This was applied to three key numeric columns: salary_usd, bonus_usd, and weekly_hours. 

def cap_outliers(df, col):
    Q1, Q3 = df[col].quantile(0.25), df[col].quantile(0.75)
    IQR = Q3 - Q1
    lo, hi = Q1 - 1.5*IQR, Q3 + 1.5*IQR
    n = ((df[col] < lo) | (df[col] > hi)).sum()
    print(f"Outliers in {col}: {n}")
    df[col] = df[col].clip(lo, hi)
    return df

df = cap_outliers(df, 'salary_usd')
df = cap_outliers(df, 'bonus_usd')
df = cap_outliers(df, 'weekly_hours')
